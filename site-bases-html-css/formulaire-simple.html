Explication

Les formulaires sont constitués d'un ensemble d'éléments. form est la balise principale, c'est grâce à elle que l'on définit le formulaire, elle contient toutes les autres balises permettant la récolte d'informations auprès de l'internaute. Elle doit obligatoirement disposer d'un attribut action indiquant à quel programme les données récoltées seront transmises.

Chaque information demandée est libellée via un élément label. La saisie en elle-même est mise en place à l'aide de la balise input pour la grande majorité des champs, la balise select pour une liste déroulante, la balise textarea pour la saisie d'un texte multilignes. L'attribut type de la balise input permet de préciser le type de saisie attendue : champ de texte, bouton radio, case à cocher, dépot de fichier, mot de passe, etc. La balise option permet de définir les entrées d'une liste déroulante. Le ou les libellés ainsi que le ou les champs de saisie liés à une même information sont regroupés, dans une même boîte, à l'aide des balises fieldset et/ou div.

Pour garantir l'accessibilité du formulaire ainsi que son ergonomie, l'élément label doit être relié au champ de saisie qu'il libelle via l'attribut for. Cet attribut doit avoir la même valeur que l'attribut id du champ de saisie (input/select/textarea).

La validation du formulaire est mise en oeuvre à l'aide d'un élément input de type submit ou image ou d'un élément button. L'information est transmise au programme receveur sous forme d'un nom et d'une valeur. Tout élément que l'on veut transmettre doit donc disposer d'un attribut name. La valeur d'un champ que l'utilisateur renseigne est fournie par cette saisie mais pour tout élément non saisissable (bouton radio, case à cocher, liste déroulante, etc) c'est à vous d'indiquer, via l'attribut value, la valeur que vous souhaitez associer à la sélection de l'élément.

La saisie de tout champ peut être rendue obligatoire grâce à l'attribut required. La validation du formulaire ne sera pas possible tant que les champs obligatoires ne seront pas renseignés. L'attribut checked permet de cocher par défaut un bouton radio ou une case à cocher, l'attribut selected permet de présélectionner une option de la liste déroulante.
